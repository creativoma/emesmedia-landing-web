# EMESMEDIA

## Description
- Social Media Management
- Agencia de medios sociales

## Resourses
- Live Sass Compiler (Extensión de VScode)
- Agregar al "settings.json" del VScode para que la extensión compile correctamente: 
  
  ``` 
    "liveSassCompile.settings.formats": [
      {
        "format": "compressed",
        "extensionName": ".css",
        "savePath": "/src/css",
      }
    ]
  ```
